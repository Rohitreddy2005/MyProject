About — Smart Traffic Management (Bhubaneswar Demo)

This prototype maps key junctions of Bhubaneswar on a live map and shows real-time traffic KPIs for each junction (queue length, signal state, last update). Traffic counts are simulated for demo. The system supports:

  • Map-based monitoring: See all major signals as markers on a map.
  • Manual control: Operators can set signal states at each junction.
  • Emergency preemption: Ambulance and VIP events trigger preemption — nearest junctions turn green for a short time.
  • Events & roadworks: Events can be added to mark blocked roads and reroute traffic.
  • Rule-based control for safety: We use explainable rules for allocation of green time now; later this will be extended to reinforcement learning.

For the jury: we do more than simple vehicle counts. We compute corridor travel / queue / throughput metrics, handle scheduled VIPs and emergencies, and show the city-level view so traffic managers can decide and intervene.
